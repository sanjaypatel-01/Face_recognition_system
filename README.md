# Face Recognition System

This repository contains **two complete face recognition pipelines**, showing the evolution from classical computer vision to deep learning–based recognition.

---

## 🔹 Versions Included

### 1️⃣ Classical Face Recognition (Haar + KNN)
📁 `classical_knn_version/`

- Haar Cascade face detection
- Raw grayscale pixel features
- KNN classifier
- Educational baseline system

👉 Best for understanding fundamentals.

---

### 2️⃣ Modern Face Recognition (FaceNet + SVM)
📁 `facenet_svm_version/`

- FaceNet deep embeddings (512-D)
- SVM classifier with probability thresholding
- Unknown face rejection
- Image, webcam & Streamlit support

👉 Production-ready, scalable approach.

---

## 🧠 Key Learning

| Aspect | Classical | FaceNet + SVM |
|------|----------|---------------|
| Feature Type | Raw pixels | Deep embeddings |
| Robustness | Low | High |
| Scalability | Poor | Excellent |
| Accuracy | Limited | ~96% |
